# 03. 保存文件

本教程将教你如何在记事本中保存和打开文件。

## 保存新文件

1. 点击左上角的 **文件** 菜单
2. 选择 **保存** 或 **另存为**
3. 在弹出的对话框中选择保存位置
4. 输入文件名（建议以 `.txt` 结尾）
5. 点击 **保存** 按钮

### 快捷键

- **保存**：按 `Ctrl + S`
- **另存为**：按 `F12`

## 选择文件编码

在保存文件时，可以在底部选择编码格式：

- **UTF-8**：推荐，支持中文和其他语言
- **ANSI**：仅支持系统默认语言
- **UTF-8 with BOM**：带 BOM 标记的 UTF-8

## 打开已有文件

1. 点击 **文件** → **打开**
2. 或直接按 `Ctrl + O`
3. 浏览并选择要打开的文件
4. 点击 **打开**
