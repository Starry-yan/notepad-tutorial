# 04. 高级功能

本教程将教你如何使用记事本的高级功能。

## 查找和替换

### 查找
1. 按 `Ctrl + F` 打开查找对话框
2. 输入要查找的文字
3. 点击 **查找下一个** 定位到匹配位置

### 替换
1. 按 `Ctrl + H` 打开替换对话框
2. 输入要查找的文字
3. 输入要替换成的文字
4. 点击 **替换** 逐个替换，或 **全部替换** 一次性替换所有

## 自动换行

1. 点击 **格式** 菜单
2. 勾选 **自动换行**
3. 文本会根据窗口宽度自动换行

> 注意：自动换行只影响显示，不会实际插入换行符。

## 字体设置

1. 点击 **格式** → **字体**
2. 选择字体、字形和大小
3. 点击 **确定** 应用

## 查看行号

Windows 11 版本的记事本支持显示行号：
1. 右键点击编辑器区域
2. 勾选 **显示行号**
