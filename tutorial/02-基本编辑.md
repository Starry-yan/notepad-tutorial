# 02. 基本编辑

本教程将教你如何在记事本中进行基本的文本编辑操作。

## 输入文本

1. 打开记事本后，直接在窗口中点击
2. 开始输入文字，就像在纸上写字一样

## 选择文本

- **选择单个字符**：将光标移到字符上，按住鼠标左键拖动
- **选择整行**：将鼠标移到行首左侧，点击左键
- **全选**：按 `Ctrl + A` 组合键

## 复制、剪切和粘贴

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 复制 | `Ctrl + C` | 复制选中的文本 |
| 剪切 | `Ctrl + X` | 剪切选中的文本 |
| 粘贴 | `Ctrl + V` | 粘贴复制/剪切的内容 |

## 撤销和重做

- **撤销**：按 `Ctrl + Z` 撤销上一步操作
- **重做**：按 `Ctrl + Y` 恢复被撤销的操作

## 查找和替换

1. 按 `Ctrl + F` 打开查找对话框
2. 输入要查找的文字
3. 点击 **查找下一个** 定位到匹配位置
