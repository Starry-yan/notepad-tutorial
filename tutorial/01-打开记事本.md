# 01. 打开记事本

本教程将教你如何在 Windows 系统中打开记事本（Notepad）软件。

## 方法一：使用搜索功能

1. 点击屏幕左下角的 **开始菜单** 或按下键盘上的 `Win` 键
2. 直接输入 "记事本" 或 "Notepad"
3. 在搜索结果中点击 **记事本** 应用

## 方法二：使用运行命令

1. 按下键盘上的 `Win + R` 组合键，打开"运行"对话框
2. 输入 `notepad`
3. 按 `Enter` 键或点击 **确定**

## 方法三：通过文件关联

1. 右键点击任意 `.txt` 文本文件
2. 选择 **打开方式** → **记事本**
